# 功能列表

* 完全支持 MQTT 3.1/3.1.1 的特性，包括 TCP/TLS/WS/WSS，即将支持 MQTT 5.0
* 原生支持多租户资源共享和工作负载隔离
* 内置存储引擎，针对关键负载定向优化，无第三方中间件依赖。
* 扩展机制支持：
    * 认证/授权 (Authentication/Authorization)
    * 运行时设置 (Runtime Setting)
    * 桥接 (Bridging)
    * 事件 (Event)
    * 系统/租户级别的监控 (System/Tenant-level Monitoring)
