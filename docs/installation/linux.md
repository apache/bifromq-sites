## Prerequisites

- JDK 17+
- Latest version BifroMQ [Releases](../get_started/download/intro.md)

## Shell Commands

```
tar -zxvf bifromq-*-standalone.tar.gz --strip-components 1  -C /usr/share/bifromq/
cd /usr/share/bifromq && ./bin/standalone.sh start
```
